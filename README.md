
- `BusinessCard.html` – The main HTML file with contact/name information.
- `styles.css` – Styles used to layout and theme the card.
- `Images/` – Folder for any logos or profile pictures.

## Getting Started

1. Clone or download the repository.
2. Open `BusinessCard.html` in a web browser.
3. Modify the HTML or CSS as desired to personalize the card.

## Customization

- Edit `styles.css` to change colours, fonts, or layout.
- Replace/add images in the `Images` folder and update the HTML `<img>` tags.
